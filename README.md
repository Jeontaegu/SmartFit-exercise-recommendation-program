🏋️‍♂️ SmartFit - 맞춤형 컨디션 기반 운동 추천 시스템

"매일 아침 변화하는 나의 생체 바이탈과 인바디 데이터를 결합한 최적의 개인 맞춤형 피트니스 처방전"

SmartFit은 모바일 사용자 환경(Mobile-first UI/UX)에 최적화된 하이브리드 운동 추천 웹 애플리케이션입니다. 사용자의 고정형 신체 스펙(인바디)과 가변형 생체 신호(바이탈 데이터)를 결합하여 당일 수행 가능한 최적의 운동 조합, 운동 강도 및 추천 시간을 실시간으로 연산해 줍니다.

📱 서비스 핵심 아키텍처 및 화면 흐름

애플리케이션은 사용자의 피로도와 접근성을 고려해 하단 탭 내비게이션 바 형식의 직관적인 디자인을 적용했습니다.

설정/목적 탭 (Setup & Goals)

운동 목적 선택: 다이어트 / 근육 증량 / 체력 증진 / 통증 및 재활

장소 선호도 필터: 전체 / 홈트레이닝(맨몸) / 헬스장(기구)

통증 및 관절 제외 필터: 무릎, 허리, 어깨, 손목, 발목, 목 등의 부상을 유발하는 운동 원천 차단

인바디 탭 (Monthly Body Metrics)

입력 주기: 월 1회 권장

입력 필드: 키, 몸무게, 골격근량, 체지방량, 체지방률

자동 연산 및 분석: BMI 자동 계산 및 BMI 지수에 따른 비만도 게이지, 골격근-체지방 균형 시각화 분석 차트 제공

생체 탭 (Daily Vital Metrics)

입력 주기: 매일 아침 권장 (웨어러블 기기 입력 대체 가능 설계)

입력 필드: 휴식기 심박수(BPM), 혈압(수축기/이완기), 혈중 산소포화도(%), 수면 시간(시간), 수면 평균 호흡수

추천운동 탭 (AI Prescription Dashboard)

바이탈 점수화(0~100점)를 통한 컨디션 판단 (최상, 양호, 피로, 경고, 위험 5단계)

안전 범위 기반의 운동 강도(상/중/하) 및 추천 시간 가이드라인 제시

맞춤형 필터링을 거친 당일 최종 운동 처방 리스트 출력

⚙️ 핵심 추천 알고리즘 설계 명세

1. 데일리 컨디션 점수 (Condition Score) 산출 공식

매일 아침 입력받는 4가지 핵심 바이탈 지표에 가중치를 부여하여 최대 100점의 점수를 계산합니다.

$$\text{Condition Score} = \text{Sleep Pts (25)} + \text{HR Pts (25)} + \text{BP Pts (25)} + \text{SpO2 Pts (25)} - \text{Penalties}$$

각 평가 부문 최대 점수: 25점

각 평가 부문의 상세 배점 기준

- 수면 시간 (Sleep)

7.5시간 이상: 25점 / 6.5시간 이상: 20점 / 5.5시간 이상: 10점 / 5.5시간 미만: -5점 감점 (수면 결핍)

- 휴식기 심박수 (HR)

60 ~ 75 BPM: 25점 / 76 ~ 85 BPM: 18점 / 86 ~ 95 BPM: 8점 / 95 BPM 초과: 2점 (과도한 각성/피로)

- 혈압 안정도 (BP)

수축기 ≤ 120 및 이완기 ≤ 80: 25점 / 수축기 < 135 및 이완기 < 88: 18점 / 수축기 ≥ 140 또는 이완기 ≥ 90: -10점 감점

- 산소포화도 (SpO2)

98% 이상: 25점 / 95% ~ 97%: 15점 / 95% 미만: -20점 감점 (저산소 위험 유발 차단)

2. 컨디션 5단계 매핑 및 운동 처방 가이드라인

점수가 산출되면 아래의 5가지 컨디션 구간으로 매핑되어 안전 운동 매트릭스가 작동합니다.

🔥 최상 (85 ~ 100점):

추천 강도: 상 (High) 및 중/하 전체 허용

운동 시간: 50 - 70 분

코치 가이드: "체내 에너지와 세포 순환이 최고 수준입니다. 안전하게 기량을 최대치로 발휘하기 좋습니다."

✅ 양호 (60 ~ 84점):

추천 강도: 중-상 (Medium - High)

운동 시간: 40 - 55 분

코치 가이드: "생체 리듬이 표준 범주에 있습니다. 평소 루틴을 부상 걱정 없이 편안히 소화할 수 있습니다."

🥱 피로 (40 ~ 59점):

추천 강도: 중-하 (Low - Medium)만 허용 (고강도 운동 자동 제외)

운동 시간: 30 - 40 분

코치 가이드: "수면이나 신경계 회복도가 저조합니다. 고중량 리프팅은 관절 부상을 야기할 수 있으므로 지양하고 가벼운 고립 위주 훈련을 하세요."

⚠️ 경고 (20 ~ 39점):

추천 강도: 하 (Low)만 허용 (고강도/중강도 운동 전체 제외)

운동 시간: 20 - 30 분

코치 가이드: "심혈관 스트레스가 높거나 수면 부족 상태입니다. 무리한 호흡 압박 운동은 피하고, 걷기나 가벼운 스트레칭으로 스트레스를 낮추세요."

🚨 위험 (20점 미만):

추천 강도: 운동 제한 / 정적 스트레칭만 노출

운동 시간: 15 분 미만

코치 가이드: "바이탈 경고 신호가 발견됩니다. 전력 스포츠나 무거운 운동은 오히려 면역을 망가트리니 완전한 휴식을 강권합니다."

3. 의료적 예방을 위한 세이프 가드 필터 (Medical Safeguard)

고혈압 비상 모드 (Hyper-tension Safeguard):

사용자의 혈압 데이터가 수축기 140 mmHg 이상 또는 이완기 90 mmHg 이상에 도달하는 즉시 적용됩니다.

복압을 급격히 상승시키고 뇌압 및 심장에 압박을 주는 무거운 다관절 프리웨이트 운동(예: 바벨 스쿼트, 루마니안 데드리프트)이 추천 리스트에서 즉각적으로 강제 격리됩니다.

관절/통증 안심 차단 필터 (Anatomy Protection Filter):

사용자가 제외하고 싶은 부위(예: 무릎, 허리, 어깨 등)를 탭하면 해당 관절에 직접적으로 부담을 주는 운동군(예: 무릎 필터 시 '바벨 스쿼트', '런닝머신' 등)이 실시간 데이터베이스 조회 단계에서 걸러집니다.

🛠️ 기술 스택 및 구현 세부사항

Front-end: HTML5, Tailwind CSS (CDN), Vanilla JavaScript (ES6+)

디자인 및 프레임워크: Mobile-First Responsive 디자인 (최대 너비 440px 고정형 디자인으로 앱 느낌 극대화)

아이콘 및 에셋: SVG 인라인 벡터 그래픽스 및 이모지 기반 경량 에셋 구성

상태 관리: Vanilla JS 전역 변수를 활용한 완전 반응형 렌더링 시스템 구축 (사용자가 탭을 전환하거나 설정을 바꿀 때 딜레이 없이 실시간 알고리즘 렌더링)

🚀 빠른 시작 (How to Run)

본 애플리케이션은 단일 파일로 동작하도록 통합 설계되었습니다. 복잡한 환경 설정이나 패키지 설치 과정 없이 즉시 실행해 볼 수 있습니다.

본 레포지토리의 index.html 파일을 다운로드합니다.

다운로드한 파일을 더블 클릭하여 크롬, 사파리 등의 웹 브라우저에서 실행합니다.

또는 다음과 같이 가벼운 로컬 개발 서버를 실행하여 스마트폰에서 실시간 테스트 환경을 구성할 수 있습니다:

# Python이 설치되어 있는 경우 간단한 서버 구동
python -m http.server 8000


스마트폰으로 동일한 Wi-Fi 네트워크에 접속한 뒤, 컴퓨터 IP주소 http://[IP주소]:8000로 접속하면 완벽한 모바일 터치 피드백을 지원하는 상태로 테스트할 수 있습니다.
